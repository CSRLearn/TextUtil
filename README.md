# 说明
项目中使用的表情图片是从其他地方拷贝出来的，尺寸大小不是很合适，原本都是@2x的，因为发送出来太小了，所以修改成了1倍图，如果要完整显示所有表情，修改图片名，去掉@2x,然后添加完全plist文件。
